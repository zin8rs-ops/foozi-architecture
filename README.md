# Product Design System

AI-native design system for scalable UI development.

## Structure

tokens/
Design tokens such as colors, spacing and typography.

components/
Reusable UI components with behavior specification.

patterns/
Higher level UI patterns such as layouts and widgets.

## Workflow

Figma → Design Tokens → Git → AI Code Generation

## Goal

Create a design system that can be read by AI coding agents to generate UI consistently.
