# Foozi Product Vision

Foozi is a football prediction and analysis platform.

Goals:

1. Make football standings and matches interactive.
2. Add prediction layers to every match.
3. Build tools for football betting insights.
4. Provide AI-generated football content.

Core experiences:

Match Page
- Match analysis
- Prediction odds
- User predictions

League Pages
- Standings
- Team performance

Prediction Game
- Predict results
- Score points
- Compete with other users
