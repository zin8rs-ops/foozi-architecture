# Data Sources

This document describes the data sources for the football platform.

---

## External APIs

The platform integrates external football data APIs.

Examples

- match data
- standings
- player statistics

---

## Scraping

Additional data may be collected from public football websites.

Examples

- market values
- transfers
- advanced statistics

---

## Internal Database

All data is stored in an internal database.

Entities

- competitions
- teams
- players
- matches
- standings

---

## Data Flow

External Data
↓
Data Ingestion
↓
Database
↓
Internal API
↓
Frontend Pages
